            Preguntas Frecuentes
          

          

            

              

                
                ¿Cómo funciona la validación de cédulas uruguayas?
              

              

                
Nuestro sistema utiliza el algoritmo oficial de verificación de cédulas uruguayas mediante el dígito verificador. Además, consulta el formulario oficial del MEF (Ministerio de Economía y Finanzas) para obtener información adicional como género y edad.

              

            

            

              

                
                ¿Es seguro usar este validador?
              

              

                
Sí, es completamente seguro. No almacenamos ningún dato personal. Las consultas se realizan directamente al servicio oficial y no guardamos historial de búsquedas ni información de usuarios.

              

            

            

              

                
                ¿Puedo usar la API en mi proyecto?
              

              

                
Absolutamente. Ofrecemos una API REST gratuita y también un paquete NPM para integrar fácilmente en proyectos Node.js. Consulta la documentación en Swagger para más detalles.

              

            

            

              

                
                ¿Qué tan rápida es la validación?
              

              

                
La validación del formato es instantánea. La consulta al formulario oficial del MEF suele tomar entre 1-3 segundos, dependiendo de la conexión y la carga del servidor del Ministerio de Economía y Finanzas.
